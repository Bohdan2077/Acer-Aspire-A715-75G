## Характеристики 
- **Екран**: <img src="icon/Screen.svg" style="vertical-align: middle; width: 35px; height: 35px;">

  - Частота оновлення: 60 Гц
  - Роздільна здатність: 1920x1080
  - Покриття: Матове
